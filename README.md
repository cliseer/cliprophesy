# cliprophesy

A CLI tool that suggests commands based on your shell history.

## Local development

* Deploying
** rm -rf build/
** python -m build
** python3 -m twine upload --repository testpypi dist/* --verbose

* Developing
** PYTHONPATH=. python main.py <command>
** Use absolute imports
